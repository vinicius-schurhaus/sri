# SRI - Semântica e Recuperação de Informação

Repositório com os notebooks e arquivos gerados para as Avaliações a Distância da disciplina de Tópicos Especiais em Computação: Semântica e Recuperação de Informação do Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas da Universidade do Estado de Santa Catarina - UDESC.

## Dataset

Dataset utilizado: [noticias-govbr-ptbr-1](https://huggingface.co/datasets/divergente/noticias-govbr-ptbr-1)

Composto por notícias institucionais do governo brasileiro em português. Foram selecionados 30 documentos para análise.
